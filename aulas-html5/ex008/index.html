Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica)

Nesta frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semantico)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos marcar o texto utilizando a tag MARK

podemos mudar a sua cor mudando o style direto no mark ou pelo head mudando todos os marks de uma vez

Texto deletado

O texto deletado seria um texto com o risquinho no meio, para indicar que ele está lá, mas não é para ser considerado, tipo esse, usando a tag DEL

Texto iserido

Seria o texto que foi inserido, sublinhado, assim ele mostra que o texto foi colocado por alguem (Semântica), apenas usando a tag INS

Existe também a tag de sublinhado com a tag U (Não semântica) que provavelmente sera obsoleta, por conta da semântica

Texto sobreescrito

Utilizado para subir a letra ou numero com a teg SUP para esse tipo de problema: x20+3

Texto subescrito

Utilizado para deixar a letra em baixo, subescrevendo ela como esse exemplo: H2O
